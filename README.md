# buzz
Репозиторий приложения для текстового и голосового общения в формате комнат.
# Пакеты, которые нужно установить
npm install tsparticles react-tsparticles

## Jitsi

For self-hosted Jitsi, set these backend environment variables before starting the API:

```env
APP_JITSI_SERVER_URL=https://jitsi.buzz.su
APP_JITSI_APP_ID=your_app_id
APP_JITSI_JWT_SECRET=your_jitsi_jwt_secret
```
